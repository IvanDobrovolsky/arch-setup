# arch-setup

Personal Arch Linux dotfiles and system setup, running [Omarchy](https://omarchy.com).

## What's included

- **Neovim** -- custom config with LSP, Treesitter, Telescope, auto-save
- **Shell** -- bash functions (`twd`, `copy`) and tmux workflow
- **Lazygit** -- auto-refresh config for tmux pane usage
- **System** -- package debloat and app installs

## Usage

```bash
git clone https://github.com/id/arch-setup.git
cd arch-setup
./setup.sh
```
